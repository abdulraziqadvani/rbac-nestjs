# Role-Based Access Control (RBAC) - PoC

## Description

This is a PoC for the Role-Based Access Control (RBAC) created on a NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# Generate Documentation
$ npm run generate:doc
```
